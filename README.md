## My Agent Skills for Dev

A collection of [opencode](https://opencode.ai) agent skills for development workflows.

### Skills

| Skill | Description |
|-------|-------------|
| **deploy-docker-production** | Build and run a local production Docker image for Next.js apps. Handles image versioning, container lifecycle, preflight checks, and port conflicts safely. |
| **git-commit-guide** | Conventional commit formatting with project-specific scopes and commit rules. |
| **project-onboarding** | Help new developers clone, set up, and run a Next.js project. Covers Prisma, Docker, and local dev setup. |

### Installation

```bash
npx skills add deploy-docker-production
npx skills add git-commit-guide
npx skills add project-onboarding
```

Or clone this repo and reference the skills from your opencode config.
